## So..., what is TIL?

> - TIL은 **T**oday **I** **L**earned의 줄임말로 개발자 사이에서 매일 자신이 학습한 내용을 **commit(기록)** 하는 것
> - github, bitbucket, gitlab과 같은 원격 저장소에서 제공하는 1 commit - 1 grass의 흥미 요소 제공
